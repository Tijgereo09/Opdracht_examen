# Voetbal Trainingsplanner

Eindproject voor Programmeren (CS50).  
Deze webapplicatie is gebouwd met Flask, SQLite, HTML/CSS en Bootstrap.

## Functionaliteiten
- Login en sessies
- Spelers beheren (CRUD)
- Trainingen plannen + aanwezigheden registreren
- Wedstrijden toevoegen
- Opstellingen opslaan per wedstrijd
- Statistieken bijhouden (goals, assists, minuten)
- Responsive design met Bootstrap

## Installatie
1. Maak een virtual environment:
   python -m venv venv
2. Activeer de venv
3. Installeer Flask:
   pip install -r requirements.txt
4. Start de app:
   flask run

## Deploy
De app wordt online gezet via PythonAnywhere.
De code staat op GitHub.
